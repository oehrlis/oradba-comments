# oradba-comments

This repository hosts the [GitHub Discussions](https://github.com/oehrlis/oradba-comments/discussions)
used as a comment backend for [oradba.ch](https://oradba.ch) via [Giscus](https://giscus.app).

## How it works

Blog posts on oradba.ch embed a comment widget powered by Giscus.
When a reader posts a comment, Giscus creates or updates a GitHub Discussion in this repository.
No account other than a standard GitHub account is required to comment.

```
oradba.ch blog post  →  Giscus widget  →  GitHub Discussion (this repo)
```

## Commenting

To leave a comment on a blog post, visit the post on [oradba.ch](https://oradba.ch/blog/)
and use the comment section at the bottom of the page.

You can also browse or participate in discussions directly here:
[github.com/oehrlis/oradba-comments/discussions](https://github.com/oehrlis/oradba-comments/discussions)

## Code of Conduct

Comments should be respectful and on-topic. Off-topic, abusive, or spam comments
will be removed. GitHub's [Community Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines) apply.
